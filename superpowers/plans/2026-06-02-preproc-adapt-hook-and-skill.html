Preprocessing Hook + preproc-adapt Skill Implementation Plan#

For agentic workers: REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (- [ ]) syntax for tracking.

Goal: Add a supported preprocess hook to deepcell_types.predict, ship a bounded preprocessing-op library inside the package, and rewrite the preproc-adapt skill to use that supported API (fixing the no-op integration bug) with install docs.

Architecture: Inference preprocessing currently lives inside the private patch_generator (resample → _percentile_threshold_nonzero → _normalize_per_channel). We add an optional preprocess callable threaded predict → PatchDataset → patch_generator; when supplied it replaces the clip+normalize step (resampling stays built-in). When None, behavior is bit-for-bit unchanged. A new deepcell_types.preprocessing_ops module provides apply_config, DEFAULT_CONFIG, and make_preprocessor, so users (and the skill) build hooks declaratively. The skill is rewritten to call predict(..., preprocess=make_preprocessor(config)).

Tech Stack: Python, NumPy, SciPy (scipy.ndimage), scikit-image; pytest; deepcell-types package (v0.1.0, unreleased).

Background facts (verified against master @ 761fd19)#

  • predict() (deepcell_types/predict.py:244) builds PatchDataset(raw, mask, channel_names, mpp, dct_config) internally; no preprocessing hook today.

  • PatchDataset.__iter__ (deepcell_types/dataset.py:133-139) calls patch_generator(self.raw, self.mask, self.mpp, dct_config=self.dct_config).

  • self.raw is already channel-masked to in-vocab channels (dataset.py:93); the aligned names are self.channel_names_standard (resolved standard marker names).

  • patch_generator (deepcell_types/preprocessing.py:311) does: transpose→rescale→_percentile_threshold_nonzero(p=dct_config.PERCENTILE_THRESHOLD)→_normalize_per_channel→pad/crop.

  • dct_config.PERCENTILE_THRESHOLD = 99.9 (config.py:225) — the real inference percentile, set by commit ec3d9f7 "fix(inference): clip at p99.9 to match the training-archive recipe". (An earlier draft of this plan said 99.0 from a stale read of the feat/immediate-cli-abstention branch — that was WRONG. The op library’s DEFAULT_CONFIG uses 99.9 and a bit-for-bit equivalence test guards it.)

  • _normalize_per_channel (preprocessing.py:246) = per-channel min-max via np.ptp, HWC layout.

  • preprocess_fov / PreprocessedFov are exported but are the archive path — the current skill’s integration target, and therefore a silent no-op for predict.

Hook contract (locked)#

preprocess(raw_resampled, channel_names) -> ndarray
    raw_resampled : float32 ndarray (C, H, W), resampled to STANDARD_MPP, channel-masked
    channel_names : list[str]  resolved standard marker names, aligned to C
    returns       : float32 ndarray (C, H, W), values in [0, 1], same C/H/W

The hook runs after resampling and channel-masking, in place of the built-in _percentile_threshold_nonzero + _normalize_per_channel. None ⇒ built-in path.

# deepcell_types/preprocessing_ops.py
"""Declarative, bounded per-FOV preprocessing ops for the ``preprocess`` hook.

Each config is a list of ``{"op": name, ...params}`` dicts applied in order to a
``(C, H, W)`` float32 array; the result must be in ``[0, 1]`` (end with
``min_max_normalize``). Build a hook for ``deepcell_types.predict`` with
``make_preprocessor(config)``.

``DEFAULT_CONFIG`` reproduces the built-in inference preprocessing
(per-channel nonzero-pixel p99 clip + min-max), so passing it is equivalent to
passing no hook at all.
"""
from __future__ import annotations

from typing import Callable, List

import numpy as np
from scipy.ndimage import gaussian_filter, median_filter

# Built-in inference percentile (mirrors DCTConfig.PERCENTILE_THRESHOLD).
_DEFAULT_PERCENTILE = 99.0

DEFAULT_CONFIG = [{"op": "clip_percentile", "p": _DEFAULT_PERCENTILE},
                  {"op": "min_max_normalize"}]


def _clip_percentile_nonzero(x, p):
    """Per-channel clip at the p-th percentile of *nonzero* pixels.

    Matches ``preprocessing._percentile_threshold_nonzero`` so ``DEFAULT_CONFIG``
    is bit-equivalent to the built-in path. All-zero channels are untouched.
    """
    out = x.copy()
    for c in range(x.shape[0]):
        nz = out[c][np.nonzero(out[c])]
        if nz.size:
            hi = np.percentile(nz, p)
            np.minimum(out[c], hi, out=out[c])
    return out


def _min_max(x):
    mn = x.min(axis=(1, 2), keepdims=True)
    ptp = np.ptp(x, axis=(1, 2), keepdims=True)
    ptp[ptp == 0] = 1.0
    return (x - mn) / ptp


def _hot_pixel_removal(x, z):
    out = x.copy()
    for c in range(x.shape[0]):
        med = np.median(out[c])
        mad = np.median(np.abs(out[c] - med)) or 1.0
        hi = med + z * 1.4826 * mad
        np.minimum(out[c], hi, out=out[c])
    return out


def apply_config(raw, channel_names, config):
    """Apply a bounded op pipeline to ``raw`` (C,H,W) -> (C,H,W) float32.

    ``channel_names`` are the resolved standard marker names aligned to ``raw``'s
    channels (what the ``preprocess`` hook receives from ``predict``).
    """
    x = np.asarray(raw, dtype=np.float32).copy()
    idx = {n: i for i, n in enumerate(channel_names)}
    for step in config:
        op = step["op"]
        if op == "clip_percentile":
            x = _clip_percentile_nonzero(x, float(step["p"]))
        elif op == "arcsinh":
            x = np.arcsinh(x / float(step.get("cofactor", 5.0)))
        elif op == "log1p":
            x = np.log1p(np.clip(x, 0, None))
        elif op == "background_subtract":
            x = np.clip(x - float(step["value"]), 0, None)
        elif op == "gamma":
            mx = x.max(axis=(1, 2), keepdims=True)
            mx[mx == 0] = 1.0
            x = np.power(np.clip(x / mx, 0, 1), float(step["g"])) * mx
        elif op == "denoise":
            kind = step.get("kind", "median")
            size = int(step.get("size", 3))
            for c in range(x.shape[0]):
                if kind == "median":
                    x[c] = median_filter(x[c], size=size)
                elif kind == "gaussian":
                    x[c] = gaussian_filter(x[c], sigma=size / 2.0)
                else:
                    raise ValueError(f"unknown denoise kind {kind!r}")
        elif op == "hot_pixel_removal":
            x = _hot_pixel_removal(x, float(step.get("z", 5.0)))
        elif op == "channel_drop":
            for n in step["names"]:
                if n in idx:
                    x[idx[n]] = 0.0
        elif op == "channel_weight":
            for n, w in step["weights"].items():
                if n in idx:
                    x[idx[n]] *= float(w)
        elif op == "min_max_normalize":
            x = _min_max(x)
        else:
            raise ValueError(f"unknown op {op!r}")
    return x.astype(np.float32)


def make_preprocessor(config) -> Callable[[np.ndarray, List[str]], np.ndarray]:
    """Return a ``preprocess`` hook for ``deepcell_types.predict`` from a config."""
    return lambda raw, channel_names: apply_config(raw, channel_names, config)
